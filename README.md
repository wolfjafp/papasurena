# 🥔 Papa Patagonia - Landing Page Premium

## 🌟 Proyecto Completamente Limpio y Optimizado

**Papa Patagonia** es una landing page profesional para una empresa de papas premium de la Patagonia chilena, desarrollada con tecnologías modernas y diseño responsive.

## 🚀 Características Principales

### ✨ Diseño Premium
- **Responsive Design**: Perfectamente adaptado para móvil, tablet y desktop
- **Navegación Intuitiva**: Menú fijo con smooth scroll y animaciones
- **Hero Section**: Impactante sección principal con call-to-actions
- **Galería Interactiva**: Visor de imágenes con modal y navegación
- **Formulario de Contacto**: Validación en tiempo real y mensajes dinámicos

### 🎨 Interfaz Moderna
- **Glassmorphism**: Efectos de transparencia y blur modernos
- **Animaciones Suaves**: Transiciones y efectos hover profesionales
- **Tipografía Clara**: Sistema tipográfico optimizado para legibilidad
- **Colores Corporativos**: Paleta basada en verdes patagónicos

### 📱 Mobile-First
- **Menú Hamburguesa**: Navegación móvil con animación
- **Botones Táctiles**: Tamaños optimizados para dispositivos móviles
- **Imágenes Responsive**: Optimizadas para diferentes pantallas
- **Performance**: Carga rápida y optimizada

```css

```

### Tipografía
- **Fuente**: System fonts (-apple-system, Segoe UI, etc.)
- **Jerarquía**: H1 (4rem) → H2 (2.5rem) → H3 (2rem)
- **Peso**: 400, 500, 600, 700, 800

### Espaciado
- **Secciones**: 5rem (desktop), 3rem (tablet), 2rem (móvil)
- **Elementos**: 1rem, 1.5rem, 2rem, 3rem, 4rem
- **Container**: 1200px máximo con padding lateral

## 📱 Breakpoints Responsive

```css
/* Mobile First */
< 768px   → Móvil (1 columna, menú hamburguesa)
768-1024px → Tablet (2 columnas)
> 1024px   → Desktop (3 columnas, layout completo)
```
